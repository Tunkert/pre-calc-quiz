Pre-Calculus Quiz 4/25/2021 Polynomials

Problem 1

Expand (a − 1/a)4 using the Binomial Theorem.

Problem 2

Expand (b + 3a)3 using the Binomial Theorem.

Problem 3

Let a(x) = 4x7 - 5x4 + 3x + 1, and b(x) = x2. Divide a by b and write the quotient as q(x) and the remainder as r(x).

Problem 4

Let a(x) = 8x6 - 4x5 + 3x2 + 4, and b(x) = 2x3. Divide a by b and write the quotient as q(x) and the remainder as r(x).

Problem 5

Let a(x) = 8x3 + 3x2 - 2x + 3, and b(x) = 2x2 + 1. Divide a by b and write the quotient as q(x) and the remainder as r(x).

Problem 6

Let a(x) = 10x5 - 7x3 + 3x2 - 7, and b(x) = 4x4 - 3. Divide a by b and write the quotient as q(x) and the remainder as r(x).
